名称 类型 默认值 描述
基本参数
init Function $.noop 初始化时调用,接受参数: opt,this指向grid本身
以下所有回调函数如果没有注明this 则默认指向grid本身
title String ''(为空) 标题,为空则不显示
iconCls String ''(为空) 标题css样式
toolBars Array false 工具栏按钮
属性:
1.text 文字
2.cls 样式
3.callBack 回调可接受参数:按钮DOM,当前按钮配置
4.disabled 默认false 是否有效
ltText String ''(为空) rowNumber表头内容
rowNumbersWidth String 0px 一般宽度可设为24px
如果设置为false,则不创建rowNumber可提高显示速度,但是列锁会无效。确定不需要列锁和rowNumber可设置为false提高展示速度。
可通过重新设置并刷新gird来动态设置这个值
rowNumbersExpand String,Boolean,Function false rowNumbersExpand : false|'auto'|模板内容|函数
false:rowNumber显示的数字都在1~pageSize之间
'auto' : rowNumber显示的数字是在当前页加pageSize的数
剩下的2个都属于自定义rowNumber内容参数
函数可接受当前行数据
模板也是当前行数据
rowNumbers2Row Boolean true click rowNumber时可触发行的click事件
showHeader Boolean true 显示列头
containerCss String datagrid-container-border 设置grid样式，默认是设置了边框
rowTpl String '' 自定义行模板
不建议使用,因为当定义了行模板时拖动列交换位置会失效
建议用rowCallBack or 单元格回调来为不同行实现不同功能
headerTpl String,jquery选择器字符串 '' 自定义列头
border Boolean true 设置边框
leftBorder Number 1 左边框大小（和css设置的数值要对应）
rightBorder Number 1 右边框大小（和css设置的数值要对应）
topBorder Number 1 上边框大小（和css设置的数值要对应）
bottomBorder Number 1 下边框大小（和css设置的数值要对应）
padding Number 8 单元格左右内边距和（和css设置的数值要对应）
width Number 自动填充容器 grid宽度，
height Number 自动填充容器 grid高度，
checkBox Boolean false 是否开启多选列，单选和多选
checkBoxWidth Number,String 28px 多选列宽度
checkBoxTitle String 列头，最好不要去修改
checkBoxFit Boolean false 多选列是否可以改变大小
editColumn Array false 编辑列
Array参数 可参考 toolBars ,callBack参数不一样，可接受的参数：t(当前按钮),rowId(行id),field('ed'),rowData(当前行数据),_item_(当前按钮配置参数)
editColumnTitle String 操作 编辑列头信息
editColumnFit Boolean true 编辑列是否可以改变大小
editCellW Number 63 编辑按钮的宽度
singleSelect Boolean false 是否只能单选
moveColumns Boolean true 列是否可拖拽交换位置
moveColumnTm Number 500 按下500ms后 会触发列拖拽效果
columns Array 空数组 表头信息,基本信息可参考_columnMetaData，eg columns : [{_columnMetaData},{_columnMetaData}]
_columnMetaData Object 
field : '',
index : '',
title : '',
width : '120px',
minWidth : 20,//最小宽度 新增
align : 'left',
_expand : false,
callBack : $.noop,
hcls : '',
bcls : '',
fcls : '',
sortable : false,， 
textLimit : false,
fitColumn : true,
disabled : false
1. field String 当前列的命名
2. index String 当前列与data数据对应字段，默认等于filed
3. title String 列头信息
4. width Number,String 当前列宽度 默认120px ,可接受 百分比，eg:以下参数是合格的: 120,'120px','30%'
5. align String 居中方式
6. _expand String(模板) 自定义单元格内容显示模板
7. callBack Function 单元格回调
8. hcls String 列头单元格样式
9. bcls String 单元格样式
10. fcls String 无效
11. sortable Boolean 排序
12. textLimit Boolean 文字溢出(即将废弃,不建议用)
13. fitColumn Boolean 当前列是否可改变大小
14. disabled Boolean 当前列是否无效
groupBy String false 对某列进行分组,eg groupBy:'title',设置后需要刷新grid
groupList Array false 分组条件,eg groupList:['2013','2014','2014'] 则对以上进行分组，默认则直接对数据分组
groupListCallBack Function $.noop 分组行 回调，接受参数 groupCell,groupName,groupData
autoScrollToField Boolean true 自动将对应列移到可视位置
autoScrollToRow Boolean true 自动将对应行移到可视位置
data Array [] grid当前页显示数据，
注：
_pk,_expand,_groupid_属于grid内部字段，不可外部进行设置
_expand字段设置后会自动触发expandRow
emptyGridMsg String '' grid显示的data为空时 grid显示的提示内容
showEmptyGridMsg Boolean true 开启emptyGridMsg
lockRows Array [] 默认需要锁的行
lockColumns Array [] 默认需要锁的列
hideColumns Array [] 默认需要隐藏的列
url String,Function '' grid数据获取地址 Function: eg: 
$(selector).extgrid({url:getGridData});
function getGridData(data,success,error){
	//获取grid数据
	//data包含分页信息
	//success(josn_data) 获取数据后成功后回调函数
	//error(msg) 获取数据后失败后回调函数
	//success,error必须要调用其中一个，
	//否则grid会处于loading
//json_data:
// {total:"255",
	rows:[{},..] [,pageSize:50,pageNumber:1,
	columns:[{}]]}	
//pageSize,pageNumber,columns是可选	
}
method String 'post' 请求类型
dataType String json 接受数据格式 目前只接受json格式数据，如果需要xml格式数据，那么需要自定义解析函数， 
eg ：
通过
...
dataType : 'xml',
init:function(){
this.xmlFilter = function(){
	//对xml处理
	return json_data;
}
}
...
loadMsg String '加载中,请稍后...' 请求中的显示信息
queryParams Object {} 请求数据
pagination Boolean false 开启分页工具栏
pagerToolBar Array false 参考toolBars
pagerMsg String 当前显示 {start} 到 {end} 条，共 {total} 条 分页栏状态信息
pageNumber Number 1 默认当前的页数
pageSize Number 10 默认当前页显示数
pageList Array [10,20,30,40,50] 可选的pageSize下拉框
sortName String '' 进行排序的列
sortOrder String 'asc' 排序类型
rowStyler String,Function '' 行样式
Function可接受参数rowid,rowdata
rowCallBack String $.noop 参考rowStyler=Function
denyRowEvents Boolean false 禁止绑定行事件，可极大提高展示速度，但是会少去很多功能，需要手动绑定事件
可编辑Grid扩展参数
type String text 输入框类型,目前支持的类型有:text,checkbox,select,date,spinner,textarea
value String '' 默认值
items Array '' select下拉框会用到，常用属性有name:'',value:'',selected:true
完整如下
cls:'',
name:'',
value:'',
readOnly:false,
disabled:false,
selected:false,
attrs:''
width+height Number '' 默认不需要设置，height对textarea有效
step Number 1 spinner控件参数
smax Number '' spinner控件参数
smin Number '' spinner控件参数
onCheck Array ['onBlur','onPaste'] 触发那些事件时候进行数据验证
disabled Boolean false 是否有效
readOnly Boolean false 是否只读
cls String '' 自定义样式
attrs String '' 自定义属性
rules String,Array '' 验证规则eg:rules:[ func,"required",{required:func},{qq:'qq..'} ],自定义函数返回false则表示验证错误
支持的验证类型:required,email,url,ip,qq,currency,mobile,phone,number,rangelength,min,max,range,equalTo
error String 'valid-error' 验证错误样式
