v1.2 BUG 发现以及修改方法:
由于发布之前没有清空测试数据，导致现在最多只能显示50行数据，请手动设置_lSize:0，修正如下： 
$(selector).extgrid({
	...
	_lSize:0,
	...
});
v1.0~v1.2 BUG 发现以及修改方法:
FireFox下行锁出现的数据不显示问题修正：
在CSS中加入以下样式：
 
.datagrid-header-outer {
	float: left;
	width: 100%;
	overflow:hidden;
	/* zoom:1;*/
}
