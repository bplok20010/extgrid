基本参数

事件名 回传参数 格式举例(没特别说明事件回调函数的this都指向grid本身) 含义 
							//没特别说明事件回调函数的this都指向grid本身
							//事件绑定1
							var grid = new dataGrid({ 
								...
								onStart:function(){},
								onStart.t:function(){},
								onStart.t2:function(){},
								...
							});
							//事件绑定2
							grid.bind( 'onStart',func );
							grid.one( 'onStart',func );
