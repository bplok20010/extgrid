基本参数

名称 参数 含义
bind eventType,func[,scope] 事件绑定
eventType(String):需要绑定的事件
func(Function):事件处理函数
scope(Object):this作用域，默认是grid本身
one eventType,func[,scope] 同bind ，区别在于只执行一次
unbind eventType,eventID 取消事件绑定
eventType(String):需要取消绑定的事件
eventID(Number):事件ID
fireEvent eventType,params 触发事件
eventType(String):需要触发的事件
params(Array):触发事件所带的参数
showLoading [msg] 显示Loading层
msg(String):提示内容
hideLoading 关闭Loading层
getColumnData [field,key,value] 设置或获取columns的配置信息,注：部分参数设置后会立即刷新表格
setColumnData [field,key,value] 设置或获取columns的配置信息,注：部分参数设置后会立即刷新表格
setColumnValue field,key,value 设置或获取columns的配置信息,注：设置后不会会立即刷新表格
getData 取得当前grid的数据
onSizeChange [width,height] 设置grid大小
setWH [width,height] 设置grid大小
onViewSizeChange 更新视图大小
getRowData rid[,isPK] 获取行数据
isPK(boolean):默认false,true:根据pk来索引
setRowData rid,field,value 设置行数据
注意field并不是column的field应该是index,但是大部分情况下field和index默认是一样的， 建议使用setFieldValue来设置单元格内容
getFieldValue rid,field 获取单元格数据容
setFieldValue rid,field,value 设置单元格数据容,return false 则无修改
getSlectRows Boolean 获取选中的行
默认是false 返回行id, 为true时返回选中的数据集
lockRow rid 锁定某行
unLockRow rid 取消某行的锁定
lockColumn field 锁定某列
unLockColumn field 取消某列的锁定
getColumns 获取列配置信息
getColumnList 获取包含所有列名的数组
showColumn field 显示某一列
hideColumn field 隐藏某一列
selectAllRows 选择所有行
unselectAllRows 取消选择所有行
selectRow rid 选择某行
unselectRow rid 取消选择某行
refreshData 刷新表格
refreshDataCache 刷新当前页
C [key,value] 设置或获取配置信息
addData Array 为grid添加数据,添加后会立即刷新表格
updateData Array 更新grid数据,更新后会立即刷新表格
deleteData Array 删除grid数据,删除后会立即刷新表格
setFieldWidth field,width 设置列宽，支持百分比
addRow rowData,ai 追加一行,ai : true默认自动识别是否超过pageSize,false:不识别直接追加一行
updateRow rid,data 更新某行
deleteRow rid 删除某行
metaData data 添加元数据:json_data
updateGrid Array 更新当前页gird数据并显示
scrollToField field 滚动到列
scrollToRow rid 滚动到某行
hideHeader 隐藏列头
showHeader 显示列头
可编辑Grid扩展API(默认this都指向grid本身)
getEditCellValue rid,field 获取编辑状态下的编辑内容
checkEditCell rid,field 验证当前编辑单元内容是否正确
editCell rid,field 编辑某单元格
saveEditCell [rid,field] 保存并验证正处于编辑状态的单元格,没指定rid,field则保存所有正处于编辑中的单元格该函数会触发onCellEdit
返回成功保存的单元格数据eg: [ {rid:1,field:'name'} ]
unEditCell rid,field 不做任何保存，以及验证，直接取消当前单元格的编辑状态
unEditCell [rid,field] 不做任何保存，以及验证，直接取消当前单元格的编辑状态,没指定rid,field则取消所有正处于编辑中的单元格
editRow rid 编辑某行
checkEditRow rid 验证当前编辑行是否有没通过的单元格
saveEditRow rid 保存当前行，返回成功验证并保存的单元格数组eg:[ {rid:1,field:'name'} ]
改函数会触发onRowEdit事件 回调函数可接受参数:rid,rowData,successSave(eg:[ {rid:1,field:'name'} ])
unEditRow rid 不做任何保存，以及验证取消当前行的编辑
